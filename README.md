# Portfolio — Fahem Aoudia

Site personnel (Next.js 15) : présentation, projets, compétences, parcours et contact. Interface **français / anglais**, déploiement adapté Vercel.

Portfolio: https://fahemaoudia.vercel.app

## Stack

- **Next.js 15** (App Router) · **React 19** · **TypeScript**
- **Tailwind CSS** · **Framer Motion**
- **Three.js** (`@react-three/fiber`, `@react-three/drei`) pour la scène du hero

## Développement local

```bash
npm install
npm run dev
```

Ouvre [http://localhost:3000](http://localhost:3000).

## Build production

```bash
npm run build
npm start
```

## Variables d’environnement (optionnel)

- `NEXT_PUBLIC_SITE_URL` — URL canonique du site (SEO, Open Graph). Ex. `https://ton-domaine.vercel.app`
